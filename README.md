# 🌍 Air Quality Forecasting from Open Monitoring Data

## 📌 Overview
This project focuses on predicting **Air Quality Index (AQI)** using open-source environmental datasets. It applies Machine Learning and Time-Series models to analyze pollution trends and generate accurate forecasts.

Air pollution is a major global concern affecting human health and the environment. This project aims to provide insights and predictions that can help in early warning systems and decision-making.

---

## 🚀 Features
- 📊 Data collection using OpenAQ API / CSV datasets
- 🧹 Data preprocessing and cleaning
- 🤖 Machine Learning models:
  - Random Forest
  - Gradient Boosting
  - LSTM / GRU (if implemented)
- 📈 Model evaluation using:
  - RMSE
  - MAE
  - MAPE
  - R² Score
- 📉 Visualization of actual vs predicted values
- 🔄 Time-series forecasting

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib  
- **Models:** Random Forest, Gradient Boosting, Deep Learning
